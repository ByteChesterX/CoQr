# QR Kod Üretici & Okuyucu

Python ile yazılmış modern arayüzlü QR kod uygulaması.

## Özellikler

- QR kod üretme (metin/URL)
- Kameradan canlı QR okuma
- Dosyadan QR okuma
- PNG/JPG olarak kaydetme
- Koyu tema arayüz

## Kurulum

curl -fsSL https://raw.githubusercontent.com/ByteChesterX/CoQr/main/install.sh | bash

## Kullanım

qr-kod

## Kaldırma

~/Projeler/qr-kod-uygulamasi/uninstall.sh

## Gereksinimler

- Python 3.10+
- tkinter
- zbar
